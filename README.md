# Frontend boilerplate

## Run application

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy application

```bash
npm run build
```

## How to add new components

```
frontent-boilerplate/
├── src/
│   ├── app/
│   │   ├── components/
│   │   |   ├── Header/
│   │   │   |   ├── Header.tsx
│   │   │       └── Header.styles.ts
│   │   |   ├── Footer/
│   │   |   │   ├── Footer.tsx
│   │   |   │   └── Footer.styles.ts
```
